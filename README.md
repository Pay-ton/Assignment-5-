# Assignment-5 Plink Ball

Welcome to my shitty attempt at recreating pinball in Unity. 

The physics don't work quite right, there are a lot of buggy hitboxes, the level design of the pegs isn't amazing...

but it works!

The basics of a game are here, and if that isn't MVP (minimum viable product) I don't know what is! 


Controls: 

Spacebar - Activate the launcher

A - Left Bumper

D - Right Bumper

That's it!
